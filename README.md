## Análisis Comparativo de Tiendas

Este proyecto tiene como objetivo analizar el desempeño comercial de cuatro tiendas que venden diferentes productos. A partir de los datos de ventas y costos, se realiza una comparación para determinar cuál de ellas presenta menor rendimiento y resulta menos conveniente mantener operativa.

## Datos Utilizados
El análisis se basa en las siguientes variables:
- **Precio**: monto de cada venta.
- **Costo de envío**: costo logístico asociado.
- **Fecha de compra**, **Vendedor**, **Lugar de compra**, entre otras columnas descriptivas.

Los cuatro datasets fueron unificados y procesados con **Python** y **Pandas** en Google Colab.

## Indicadores Analizados
Para cada tienda se calcularon:
- **Ventas Totales**
- **Ticket Promedio** (promedio del valor por venta)
- **Cantidad de Ventas**
- **Costo Promedio de Envío**

## Resultados Principales
Los indicadores muestran que:
- La **Tienda 4** presenta la **menor facturación total** y el **ticket promedio más bajo**.
- Si bien posee el **costo de envío más bajo**, esta ventaja no compensa el bajo nivel de ingresos generados.

## Conclusión
Se recomienda **cerrar o reconvertir la Tienda 4**, ya que su desempeño comercial es significativamente inferior al de las otras tiendas. Los recursos destinados a su operación pueden reorientarse para optimizar la rentabilidad general.
